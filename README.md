## 🎉 Começando

Clone o repositório:

```bash
git clone https://github.com/MattZ6/nodejs-code-fix-challenge
```

Instale as dependências do projeto:

```bash
yarn
```

## 🔥 Executando

Para executar a aplicação em ambiente de desenvolvimento (localmente):

```bash
yarn dev
```

## 🧪 Testes

Para executar os testes:

```bash
yarn test
```
